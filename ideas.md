## Концепция 1: "Cinematic Noir" (Кинематографический нуар)

**Design Movement:** Film Noir meets Editorial Design. Вдохновлено кинематографической эстетикой темных тонов с драматическим освещением, как кадры из фильмов Финчера.

**Core Principles:**
1. Драматический контраст: глубокий черный фон (#0A0A0A) с точечным золотым светом
2. Кинематографические пропорции: широкие горизонтальные секции, как кадры из фильма (16:9)
3. Минимализм деталей при максимуме эмоций: каждый элемент на странице "заслуживает" свое место
4. Текстурность: зернистость пленки, мягкие тени, глубина

**Color Philosophy:** Основа - почти чистый черный (#0A0A0A) с теплым золотом (#C9A96E) как акцент, символизирующий роскошь и премиальность. Белый (#F5F0EB) для текста, создающий ощущение "света на темной сцене". Приглушенный бежевый (#D4C5B0) для вторичных элементов.

**Layout Paradigm:** Полноэкранные секции с ассиметричным расположением контента. Hero занимает 100vh с диагональным разделением: фото слева (60%), текст справа (40%). Секции чередуются: полноширинные изображения и компактные текстовые блоки с большими отступами.

**Signature Elements:**
1. Горизонтальные золотые линии-разделители, которые анимированно "рисуются" при скролле
2. Эффект "кинопленки" - легкая зернистость на фоновых изображениях
3. Крупные номера секций (01, 02, 03) в стиле editorial-журналов

**Interaction Philosophy:** Медленные, кинематографические переходы. Элементы появляются как кадры в фильме - плавно, с задержкой. Hover-эффекты раскрывают скрытые детали, как свет в темной комнате.

**Animation:** Parallax на фоновых изображениях (медленный, 0.3x). Fade-in с translate снизу для текстовых блоков (duration: 1.2s, ease: cubic-bezier). Золотые линии рисуются слева направо при попадании в viewport. Счетчики цифр анимируются с easing.

**Typography System:** Заголовки: Cormorant Garamond (Light/SemiBold) - элегантная антиква с характером. Текст: Outfit (Light 300 / Regular 400) - современный геометрический гротеск. Акценты/номера: Space Grotesk (Medium) - технический, но стильный.

## Концепция 2: "Warm Industrial" (Теплый индустриальный)

**Design Movement:** Industrial Chic meets Hospitality Warmth. Сочетание сырой индустриальной эстетики (как лофт-площадки для мероприятий) с теплотой гостеприимства.

**Core Principles:**
1. Контраст текстур: гладкие поверхности рядом с "сырыми" элементами
2. Теплая палитра на холодной основе: стальной серый + терракота + кремовый
3. Функциональная красота: каждый элемент выглядит "рабочим", но эстетичным
4. Открытое пространство: много воздуха, как в лофте с высокими потолками

**Color Philosophy:** Основа - глубокий графитовый (#1C1C1E) с теплым терракотовым акцентом (#C4704B), символизирующим огонь, тепло, гостеприимство. Кремовый (#F7F3ED) для контента. Стальной (#3A3A3C) для вторичных элементов.

**Layout Paradigm:** Сетка с "разрывами" - элементы намеренно выходят за границы колонок, создавая ощущение динамики. Левая навигация-полоска с номерами секций. Карточки услуг расположены в шахматном порядке. Фотографии "вырываются" из контейнеров.

**Signature Elements:**
1. Тонкие вертикальные линии-навигаторы по бокам страницы
2. Фотографии с "рваными" краями или маской в форме неправильного многоугольника
3. Крупные буквы-водяные знаки (ALLRENT) на фоне секций

**Interaction Philosophy:** Энергичные, но контролируемые. Карточки при наведении слегка "приподнимаются" с тенью. Фотографии при скролле двигаются с разной скоростью. Кнопки имеют "упругий" эффект при клике.

**Animation:** Staggered reveal для карточек (каждая следующая с задержкой 0.15s). Масштабирование фотографий при скролле (scale 1.05 -> 1.0). Горизонтальный параллакс для фоновых текстур. Пружинистые hover-анимации (spring physics).

**Typography System:** Заголовки: Bebas Neue (Regular) - мощный, индустриальный, кричащий. Текст: Source Sans 3 (Light/Regular) - читаемый, нейтральный, профессиональный. Акценты: Archivo (SemiBold) - геометрический, уверенный.

## Концепция 3: "Velvet Evening" (Бархатный вечер)

**Design Movement:** Art Deco Revival meets Contemporary Luxury. Вдохновлено эстетикой великих вечеров: блеск хрусталя, мягкость бархата, геометрия Ар-Деко.

**Core Principles:**
1. Тактильная роскошь: дизайн должен "ощущаться" дорогим, как прикосновение к бархату
2. Геометрическая элегантность: четкие линии, симметрия с намеренными нарушениями
3. Световые акценты: элементы "светятся" на темном фоне, как свечи на банкетном столе
4. Ритм и повторение: визуальный ритм, как музыка на вечернем приеме

**Color Philosophy:** Глубокий темно-синий (#0D1B2A) как основа - цвет вечернего неба. Мягкое шампанское-золото (#D4AF37) - блеск бокалов. Пыльная роза (#B76E79) - теплый женственный акцент. Слоновая кость (#FFFFF0) для текста.

**Layout Paradigm:** Вертикальная ось симметрии с "разлетающимися" элементами. Hero с центрированным текстом и декоративными геометрическими рамками в стиле Ар-Деко. Секции разделены декоративными SVG-паттернами. Карточки в сетке 3 колонки с "парящим" эффектом.

**Signature Elements:**
1. Геометрические Ар-Деко рамки и уголки вокруг ключевых элементов (тонкие золотые линии)
2. Градиентные "свечения" (glow) за ключевыми элементами - как свет свечей
3. Декоративные SVG-разделители между секциями с геометрическим паттерном

**Interaction Philosophy:** Утонченные и "дорогие". Hover создает мягкое свечение вокруг элемента. Кнопки имеют эффект "наполнения" цветом слева направо. Фотографии при наведении слегка увеличиваются с виньетированием.

**Animation:** Мягкий fade-in с scale (0.95 -> 1.0) для секций. Золотые декоративные элементы "проявляются" с opacity-анимацией. Счетчики с плавным нарастанием. Параллакс на декоративных геометрических элементах. Subtle shimmer-эффект на золотых акцентах.

**Typography System:** Заголовки: Playfair Display (Bold/Black) - классическая элегантность с характером. Текст: Raleway (Light 300 / Regular 400) - изящный, воздушный гротеск. Акценты/CTA: Montserrat (SemiBold 600) - уверенный, современный.
